# Gmail Safe

## Requirements
- Java 11

## Commands

### Build Project

#### Windows:
```  
gradlew.bat build  
```  
#### OS X
```  
./gradlew build  
```

### Run Service
Before run the service, it is necessary to update the file **credentials.json**  with your own credentials generated from Google account.

#### Windows:
```  
gradlew.bat bootRun  
```  
#### OS X
```  
./gradlew bootRun  
```
